FortCopen

Firewallový štít pro Minecraft servery

A firewall shield for Minecraft servers

Proč FortCopen vznikl?

Stal jsem se obětí projektu Copenheimer, což byla organizovaná skupina zaměřená na útoky na Minecraft servery. Rozhodl jsem se vytvořit nástroj, který automaticky blokuje neautorizované IP adresy a zároveň upozorňuje administrátory pomocí Discord webhooků.

Bezpečnost a neoficiální klienti

Pokud si hostujete vlastní Minecraft server, nebo ho provozuje někdo jiný, doporučuji důkladně zabezpečit přístup a přidat další vrstvu ochrany — například pomocí aplikace FortCopen. To platí především v případě, že umožňujete připojení hráčům bez zakoupeného Minecraftu (tzv. offline režim). V takovém případě se může kdokoliv připojit pod libovolným jménem a z jakékoli IP adresy, což zvyšuje riziko útoků a zneužití. FortCopen pomáhá chránit váš server i v těchto scénářích.

Co FortCopen umí?

  • ⛔ Detekuje pokusy o připojení mimo whitelist
  • 🧱 Automaticky přidává pravidla do Windows Firewallu
  • 📡 Posílá upozornění na Discord s IP a jménem hráče
  • 🖥️ Nabízí přehledné grafické rozhraní pro správu IP adres
  • 🔐 Ovládání pomocí rcon portu
  • 💾 Automatická záloha světa

Jak začít?

Stáhni si nejnovější verzi, spusť ji jako správce a nastav whitelist a port. Pokud chybí Npcap, FortCopen se postará o instalaci.

Why was FortCopen created?

I was a victim of the Copenheimer project — a coordinated group attacking Minecraft servers. I decided to create a tool that automatically blocks unauthorized IPs and alerts administrators using Discord webhooks.

Security and Offline Clients

If you are hosting your own Minecraft server, or someone is hosting it for you, it's highly recommended to properly secure access and add an extra protection layer — like FortCopen. This is especially important if your server allows connections from players without a purchased Minecraft account (i.e., in offline mode). In such cases, anyone can connect using any username and from any IP address, significantly increasing the risk of abuse or attacks. FortCopen helps secure your server even in these situations.

What can FortCopen do?

  • ⛔ Detects connection attempts outside the whitelist
  • 🧱 Automatically adds Windows Firewall rules
  • 📡 Sends alerts to Discord with IP and player name
  • 🖥️ Provides a simple GUI to manage IPs and ports
  • 🔐 Control via rcon port
  • 💾 Automatic world backup

Getting Started

Download the latest version, run it as administrator, and configure the whitelist and port. If Npcap is missing, FortCopen will handle the installation automatically.
